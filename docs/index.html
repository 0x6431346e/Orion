Siderus Orion

Orion is an Open Source easy to use Inter Planetary File System (IPFS) desktop client. It is designed to help the first-time users to join the decentralised IPFS network and to interact with dApps and modern browsers without technical knowledge.

Download GitHub Subscribe

Features

  • Designed to be simple and easy to use,
  • Works with Windows, macOS and Ubuntu Linux,
  • Everything included: no other dependencies required, no terminal needed!
  • Works with your IPFS Browser Companion,
  • Works with most dApps and Ethereum Blockchain,
  • Explores both IPFS and IPNS objects,
  • Works with Siderus Hera,
  • Customizable configuration for developers.
